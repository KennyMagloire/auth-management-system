# Authentication Management System

## Overview


In this project,  a simple authentication system was developped using **Java Swing** and **SQLite**.
The user will be able to create an account, register it, then log in using the provided credential.

## Features
- User registration
- User Login
- Database Integration
- Input validation

## Instruction to run the project
- Clone the repository
- Open the project in an IDE
- ADD the SQLite JDBC driver
- Run the program
